# Lenta 2025圣诞自留版数据处理

## 任务概述
将自留版Excel文件的数据按照模板Excel进行填充,生成3个结果文件(塑料球、外厂、陶瓷)。

## 数据映射规则

### 模板Sheet1 (TOTAL PREORDER) - 从第12行开始
| 源列 | 目标列 | 处理规则 |
|------|--------|----------|
| B列(索引1) | N列(14) | 图片(使用WPS DISPIMG公式,无法提取) |
| D列(索引3) | P列(16) | 直接复制 |
| F列(索引5) | U列(21) | 直接复制 |
| N列(索引13) | AV列(48) | 直接复制 |
| AD列(索引29) | BV列(74) | 乘以10 |
| A列(索引0) | BW列(75) | 乘以10 |
| AF列(索引31) | BX列(76) | 乘以10 |
| AL列(索引37) | BY列(77) | 乘以1000 |
| AK列(索引36) | BZ列(78) | 乘以1000 |

### 模板Sheet2 (DC) - 从第10行开始
| 源列 | 目标列 | 处理规则 |
|------|--------|----------|
| N列(索引13) | D列(4) | 直接复制 |

## 数据来源
- 自留版文件: Lenta 2025圣诞自留版.xlsx
  - 包含3个sheet: 塑料球(22行数据)、外厂(9行数据)、陶瓷(15行数据)
  - B列使用WPS的DISPIMG公式嵌入图片,openpyxl无法处理
  - 外厂sheet有600个图片对象但位置不正确

- 模板文件: （Lenta）Preorder. Эталонная форма.xlsx
  - 包含3个sheet: TOTAL PREORDER, DC, SP
  - 需要填充前两个sheet

## 生成的文件
1. 塑料球_填充结果.xlsx
2. 外厂_填充结果.xlsx
3. 陶瓷_填充结果.xlsx

## 注意事项
1. WPS的DISPIMG公式无法被openpyxl处理,图片无法提取和填充
2. 外厂sheet有图片对象但位置混乱,可能需要手动调整
3. 陶瓷sheet只有第3行有数据,其他都是空行(这是正常的)
4. 所有文本数据已按规则正确填充
5. 新增行复制了模板的格式
